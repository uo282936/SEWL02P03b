Las instrucciones básicas están incluídas en el propio html, en párrafos y demás

A la aplicación se le debe cargar un archivo de texto plano, que debe contener información relativa
a ciertas figuras, teniendo un formato fijo cada figura. Este formato está especificado en el html,
para que el usuario sea conocedor de este hecho.

Si se carga el archivo correctamente, se mostrará un botón de pintar, que leerá todas las figuras
que han sido declaradas en el fichero de entrada y las reconstruirá gráficamente sobre el lienzo.

Finalmente, si el usuario desea copiar el contenido del fichero que ha generado dicha imagen, simple-
mente debe dar al botón de "Copiar a portapapeles", y se le guardará el contenido de este. Se deja
habilitado un area de texto para que se pueda pegar el contenido del portapapeles y comprobar así que
se haya copiado correctamente.

SE DISPONEN DE DOS ARCHIVOS DE TEXTO PLANO DE EJEMPLO, BIEN FORMATEADOS, PARA CARGAR A LA APLICACIÓN Y
OBSERVAR QUE MUESTRAN LO ESPERADO. RECIBEN EL NOMBRE DE: ejemplo1.txt, ejemplo2.txt
